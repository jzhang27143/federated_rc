class federatedrc.server.FederatedServer(model_class, configpath='', interactive=False, verbose=False, listen_forever=True, n_clients=None)¶

Bases: object

Server in the Federated System.

Parameters

  • model_class (nn.module) – Pytorch model to run the Federated System on.

  • configpath (String) – File path to the server configuration.

  • interactive (Boolean) – Indiciates whether the server will allow for command line interface.

  • verbose (Boolean) – Inicates whether the server will print verbose comments.

  • listen_forever (Boolean) – Indicates whether server waits for a specified number of clients.

  • n_clients (Int) – The number of clients the server will wait to connect. Must be defined if listen_forever is false.

configure()¶

Fetches server config file to setup the FederatedServer.

run()¶

Runs the Federated Server. Waits for the specified number of clients to cnnect, or wait forever if listen_forever is true.

start_federated_averaging()¶

Begins the federated averaging process.
